1.What is Html?

  • HTML, which stands for HyperText Markup Language, is the standard markup language used to create web pages
  • It provides the structure and content of a webpage, defining how text, images, and other elements are displayed in a browser.
  • HTML is not a programming language but a markup language, meaning it uses tags to annotate text and define its structure and presentation.

5.What are lists in Html?

  • In HTML, a list is a structural element used to organize and present information in a clear, readable format. HTML provides three main types of lists.
  • Unordered Lists : These lists are used when the order of the items is not important. Each list item is typically marked with a bullet point.
  • Ordered Lists : These lists are used when the order of the items is significant. Each list item is typically marked with numbers, letters, or Roman numerals, by default.
